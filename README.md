# PyTest-Mini-Projects

## Description
A beginner-friendly demonstration of the Software Testing Life Cycle (STLC)
using Python and Pytest.

---

## Projects Included

### 1. Calculator
- Basic arithmetic operations (add, subtract, multiply, divide)
- Edge case: divide by zero

### 2. User Authentication
- Valid/invalid credentials
- Empty field validation
- Password length check
- Case sensitivity check

### 3. Debug Demo
- Identifying and fixing ZeroDivisionError
- Fixing wrong data types (set vs list)

---

## Tech Stack
- Python 3.13
- Pytest
- Git & GitHub
- VS Code

---

## How to Run
# Clone the repo
git clone https://github.com/isyakirah7/PyTest-Mini-Projects.git

# Create virtual environment
python -m venv .venv
.venv\Scripts\Activate.ps1

# Install dependencies
pip install pytest

# Run tests
pytest -v
